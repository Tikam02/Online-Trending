FAQ

  1. What is “trends”?
    Trends is a source for what's new and trending on the web. We provide content from different platforms in different categories ranging from health to news.
  2. How often does the website fetch trending articles?
    Every 30 minutes, articles are fetched from different platforms. So the front page is constantly in motion and (hopefully) filled with fresh, interesting links.
  3. Can anyone bookmark an article?
    No, anyone cannot bookmark an article. Only register users are allowed to use this feature.s
  4. Do I have to register to access all the categories?
    No, any guest user, as well as a registered user, can access all the categories.
  5. Can I reset the password?
    Yes, you can reset your password using the change password option on the right-most corner of your screen.
  6. What if I forgot password?
    You can change your password using forgot password link.
  7. Does this website have any provision to avoid eye strain during night time?
    Yes, we have provided with day/night mode to switch as per convenience.
  8. How to bookmark an article ?
    To bookmark any article, click on the blue star on the left side of the article flex card.
  9. Who can use the search feature to search the articles ?
    Any user (Register/Unregister) can use this feature.
